Q: Bootstrap Forms?
= 1. Bootstrap Vertical Form (Default) 
   (.form-control) class
  2. Bootstrap Inline Form
   (.form-inline) class [This only applies to forms within viewports that are at least 768px wide!]
  3. Bootstrap horizontal Form
   A. (.form-horizontal) class to the <form> element
   B. (.control-label) to all <label> elements
  4. There might be a situation when you need to place just plain text next to a form label instead of a form control. You can do this within a horizontal form layout by using the 
     .form-control-static class on a <p> element
